Instruksione për Hostimin në GitHub Pages

Udhëzuesi i mëposhtëm ju ndihmon të hostoni faqen e shkollës "Kenan Halimi" në GitHub Pages.

Hapi 1: Krijoni një llogari në GitHub

Nëse nuk keni ende një llogari GitHub:

  1. Shkoni në https://github.com
  2. Klikoni në butonin "Sign up" dhe ndiqni udhëzimet për të krijuar një llogari të re

Hapi 2: Krijoni një depo të re (repository)

  1. Pasi të keni hyrë në llogarinë tuaj, klikoni në butonin "+" në këndin e sipërm djathtas të ekranit
  2. Zgjidhni "New repository"
  3. Te "Repository name", shkruani: shkolla-fillore-kenan-halimi
  4. Mund të shtoni një përshkrim opsional: "Faqja zyrtare e shkollës fillore Kenan Halimi"
  5. Zgjidhni "Public" si opsion për vizibilitetin
  6. Klikoni në "Create repository"

Hapi 3: Lidhni repositoryn lokale me GitHub

Hapni terminale (PowerShell) dhe ekzekutoni komandat e mëposhtme:

cd "d:\shkolla fillore Kenan Halimi"

# Zëvendësoni "yourusername" me emrin tuaj të përdoruesit në GitHub
git remote add origin https://github.com/yourusername/shkolla-fillore-kenan-halimi.git

# Dërgoni të dhënat në GitHub
git push -u origin master

Pas ekzekutimit të këtyre komandave, do t'ju kërkohet të identifikoheni në GitHub.

Hapi 4: Aktivizoni GitHub Pages

  1. Shkoni në depon tuaj në GitHub
  2. Klikoni në "Settings" në menunë e sipërme
  3. Lëvizni poshtë deri te seksioni "GitHub Pages"
  4. Te "Source", zgjidhni "master branch" nga menuja drop-down
  5. Klikoni "Save"
  6. Faqja do të publikohet dhe do t'ju shfaqet një link në formatin: https://yourusername.github.io/shkolla-fillore-kenan-halimi/

Mund të duhen disa minuta që faqja të bëhet e disponueshme.

Shënim i rëndësishëm

Sigurohuni që keni vendosur saktë emrin e përdoruesit tuaj të GitHub në komandën e mësipërme. Për shembull, nëse emri i përdoruesit tuaj është "shkollakenanhalimi", atëherë komanda duhet të jetë:

git remote add origin https://github.com/shkollakenanhalimi/shkolla-fillore-kenan-halimi.git

Hapi 5: Përditësimi i faqes

Kur bëni ndryshime në faqen tuaj, mund t'i ngarkoni me këto komanda:

git add .
git commit -m "Përshkrimi i ndryshimeve"
git push

Ndryshimet do të pasqyrohen në faqen e publikuar pas disa minutash.
